1. Text Preprocessing

Raw text is noisy. We implemented a cleaning pipeline using NLTK and Regex:

def clean_text(text):
    text = text.lower()                 # Normalize case
    text = re.sub(r"http\S+", "", text) # Remove URLs
    text = re.sub(r"[^a-z\s]", "", text)# Keep only letters
    # Remove stopwords (e.g. "the", "is", "in")
    tokens = [t for t in text.split() if t not in STOP_WORDS]
    return " ".join(tokens)

2. Feature Extraction

We compared two ways to turn text into numbers:

Bag of Words (BoW)

Simply counts how many times each word appears. Ignores order.

TF-IDF (1-2 grams)

Weighs words by uniqueness. Includes bigrams (e.g., "Prime Minister").

üìù Conclusion & Limitations

This project demonstrates that for highly distinct topics, simple methods often outperform complex ones. However, our small dataset (159 docs) likely led to overfitting in some TF-IDF models. In a real-world scenario, we would need thousands of documents to truly validate these metrics.
